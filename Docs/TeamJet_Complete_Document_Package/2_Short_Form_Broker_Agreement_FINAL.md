# SHORT-FORM BROKER AGREEMENT

## Executive Summary and Term Sheet

**Sports Media, Inc. and Chief Executive Air**

**Date:** January 24, 2026

---

## PURPOSE

This Short-Form Agreement summarizes the principal commercial terms governing the charter brokerage partnership between Sports Media, Inc. ("Sports Media") and Chief Executive Air ("Broker"). This document incorporates the Full Master Services Agreement ("MSA") by reference. In the event of any conflict, the Full MSA shall control.

## KEY COMMERCIAL TERMS

### Term and Renewal

| Element          | Terms                                 |
| ---------------- | ------------------------------------- |
| Initial Term     | Five (5) years from Effective Date    |
| Renewal Option   | One (1) additional five (5) year term |
| Renewal Notice   | 90 days prior to expiration           |
| Maximum Duration | Ten (10) years total                  |

### Exclusivity

| Element                | Terms                                             |
| ---------------------- | ------------------------------------------------- |
| Launch Status          | Exclusive charter brokerage partner               |
| Performance Protection | Secondary broker allowed if SLAs missed           |
| SLA Trigger            | 3 urgent misses/30 days OR 5 total misses/60 days |
| Cure Period            | 15 business days to restore performance           |

### Compensation Structure

| Fee Type                   | Amount        | Recipient           |
| -------------------------- | ------------- | ------------------- |
| Turboprop Fixed Fee        | $1,250/flight | Sports Media        |
| Light Jet Fixed Fee        | $1,500/flight | Sports Media        |
| Midsize Jet Fixed Fee      | $1,750/flight | Sports Media        |
| Super-Midsize Fixed Fee    | $2,250/flight | Sports Media        |
| Heavy Jet Fixed Fee        | $3,000/flight | Sports Media        |
| Ultra-Long Range Fixed Fee | $3,500/flight | Sports Media        |
| VIP Airliner Fixed Fee     | $5,000/flight | Sports Media        |
| Standard Platform Fee      | $500/booking  | Sports Media (100%) |
| Urgent Platform Fee        | $750/booking  | Sports Media (100%) |

### Quarterly Bonuses

| Volume Threshold       | Bonus Amount |
| ---------------------- | ------------ |
| 25+ Completed Flights  | $10,000      |
| 50+ Completed Flights  | $25,000      |
| 100+ Completed Flights | $50,000      |

### Retainer

| Element           | Terms                                         |
| ----------------- | --------------------------------------------- |
| Amount            | $5,000                                        |
| Recovery          | 100% via $250 offset per flight               |
| Expected Recovery | First 20 completed flights                    |
| Forfeiture        | Early termination by Broker (first 12 months) |

## KEY PROTECTIONS

### For Sports Media

- Non-circumvention clause (24 months post-termination)
- Right to secondary broker if SLAs missed
- Agency of Record for advertising/sponsorship opportunities
- 100% retention of platform and subscription fees
- Performance-based exclusivity

### For Chief Executive Air

- Exclusive launch partnership
- 5-year initial term with renewal option
- 100% recoverable retainer
- Cure period before secondary broker activation
- Clear SLA definitions and measurement
